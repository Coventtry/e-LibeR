Requisitos Previos antes de comenzar

Asegúrate de tener lo siguiente instalado en tu computadora:


1. XAMPP (incluye Apache y MySQL).
2. Un navegador web (como Chrome o Firefox).
3. Un editor de texto o IDE (opcional, como Visual Studio Code).

e-LibeR es un sistema de gestión bibliotecaria diseñado para facilitar la administración de libros, socios y préstamos.
Ideal para pequeñas bibliotecas, este proyecto se desarrolla utilizando PHP y MySQL y se ejecuta mediante XAMPP.


Pasos para Descargar e Implementar e-LibeR

1. Descargar el Repositorio
Ve a este repositorio en GitHub.
Haz clic en el botón Code y selecciona Download ZIP.
Extrae el contenido del archivo ZIP en tu computadora.

2. Configurar XAMPP
Inicia XAMPP y activa los módulos Apache y MySQL.
Copia la carpeta del proyecto (descargada previamente) en la ruta:
C:\xampp\htdocs
La carpeta del proyecto debería llamarse, por ejemplo, e-libeR.

3. Importar la Base de Datos
Abre tu navegador y dirígete a:
http://localhost/phpmyadmin
Crea una nueva base de datos llamada biblioteca.
En la base de datos recién creada, haz clic en Importar.
Selecciona el archivo biblioteca.sql que se encuentra en la carpeta del proyecto.
Haz clic en Continuar para finalizar la importación.

4. Configurar el Archivo de Conexión a la Base de Datos
Abre el archivo de configuración de conexión (por ejemplo, config.php) en la carpeta del proyecto.
Verifica que los datos de conexión coincidan con los de tu XAMPP:
$host = 'localhost';
$user = 'root';
$password = '';
$database = 'biblioteca';

5. Ejecutar el Proyecto
Abre tu navegador y dirígete a:
http://localhost/e-libeR
Ahora deberías ver el sistema en funcionamiento.
